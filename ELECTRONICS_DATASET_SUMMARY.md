## Data Attributes

Each product record includes:

| Attribute | Description | Example |
|-----------|-------------|---------|
| **product_id** | Unique product identifier | 1, 2, 3... |
| **product_name** | Full product name | "Apple iPhone 14 Pro Max" |
| **category** | Electronics category | "Smartphones" |
| **brand** | Product brand | "Apple" |
| **model** | Product model | "iPhone 14 Pro Max" |
| **price** | Product price in INR | 139000 |
| **rating** | Average rating (1-5) | 4.5 |
| **warranty_years** | Warranty period in years | 1, 2, 3, 5 |
| **website** | E-commerce website | "Amazon", "Flipkart", etc. |
| **product_link** | Direct product link | "https://amazon.in/s?k=..." |
| **in_stock** | Stock availability | "Yes", "No" |
| **description** | Product description | "Premium product with features..." |
